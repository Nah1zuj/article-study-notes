# Source Quality And Citation Rules

Use scholarly discussion to position and explain the material. Do not use external sources to decorate the note.

In notes mode, scholarly supplementation is recommended for complex, broad, or controversial materials, but ask the user before using web/database scholarship unless they already requested it. Aim for 4-6 high-weight, verifiable academic sources when available and approved, but treat that range as a target rather than a quota. Quality overrides count: do not use encyclopedias, blogs, media pieces, vendor pages, or search snippets merely to fill the target.

## Source Priority

High priority:

- user-provided source material
- locally supplied primary course materials
- user-designated primary texts, textbooks, monographs, articles, legal texts, datasets, or reading packets
- user-added local secondary-literature folders
- peer-reviewed journal articles
- academic books and book chapters
- university press publications
- authoritative handbooks and companions
- discipline-specific encyclopedias such as SEP or IEP for philosophy
- textbooks when they are appropriate for background

Medium priority:

- working papers from reputable institutions
- official reports from universities, labs, think tanks, or public agencies
- conference papers in fields where they are primary venues, such as AI
- documentation or technical reports for technical systems when they are primary evidence

Low priority:

- general encyclopedias
- blogs
- media articles
- vendor white papers
- unsourced slide decks
- search snippets

Low-priority sources may provide leads or context, but should not anchor scholarly claims.

## Citation Rules

- Put external scholarly sources in a references section titled in the chosen note language.
- Clearly mark which references are user-provided/local and which were externally supplemented when external scholarship is used.
- If external scholarship was not used, add a brief note such as `No external scholarly sources were added; this note is based on the supplied material.` in the references or source note area.
- Use a consistent format within a note.
- Include author, year, title, publication venue or publisher, and URL/DOI when available.
- Integrate external scholarship naturally into the relevant concept, argument, textual-basis, method, evidence, or dispute node.
- Use in-prose attribution when a view belongs to a named scholar, school, institution, or report.
- Do not force frequent parenthetical attribution for ordinary background synthesis when the reference list is enough.
- Do not invent page numbers.
- Do not invent citations from memory.
- Do not cite search snippets as if they were full sources.
- If precise bibliographic data cannot be verified, say so briefly or omit the source.
- For primary-text quotations, locate the work and preferably the chapter, section, page, paragraph, statute, proposition, formula, table, figure, or another stable locator. If exact location cannot be verified from local material or reliable lookup, paraphrase rather than quote.
- When a reliable page number is available from a local PDF, page references may be used for the user's benefit; never infer page numbers from memory or OCR line positions.
- Choose source language by disciplinary context and the user's output language.

## In-Prose Attribution

Use in-prose attribution when attribution clarifies ownership of a claim:

```markdown
Rawls's contractarian framing treats justice as a problem of fair terms of cooperation (Rawls, 1971).
```

For ordinary background synthesis, a reference list is often enough.

## Local Source Hierarchy

When multiple local sources address the same issue, respect the user's stated hierarchy. If no hierarchy is stated:

1. Use the user-provided main source as the controlling source.
2. Use course materials or slides to define boundaries, examples, and emphasis.
3. Use a textbook, monograph, article, statute, dataset, or reading packet to organize the internal conceptual or argumentative order when it is more coherent than slide order.
4. Use local secondary literature to clarify disputes, interpretations, context, methods, or evidence.
5. Use external scholarship for verification, background, and representative debates only after user confirmation, unless already requested.

Write the final claim directly in scholarly prose rather than reporting each source in sequence. The source relation is internal to the workflow unless attribution improves understanding.

## Primary-Source Quotation Pattern

Use this pattern only when the wording matters:

```markdown
> Short primary-source passage. (Verified location)

Explanation: explain what the passage establishes, distinguishes, proves, complicates, or motivates.
```

Do not quote primary text as a decorative motto. Quote only when the wording matters for the concept, argument, evidence, or interpretive problem.

## Forbidden Behaviors

- Do not write "scholars generally agree" unless the source base supports it.
- Do not cite search snippets as if they were full sources.
- Do not blend AI inference into "scholarly discussion."
- Do not use external discussion to override the source text without explanation.
- Do not write final notes as a source ledger. Avoid `the slides say`, `the textbook mentions`, `according to the PDF`, `after checking`, `OCR`, `pending`, and similar process markers unless the user asks for an audit.
